# Sales Analysis (Superstore Dataset)

## Overview
Business-focused exploratory data analysis on a Superstore sales dataset to understand sales trends, profitable regions, and product performance.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Monthly Sales Trend
- Top 10 Products by Sales
- Profit by Region
- Top 10 Loss-Making Products

## Outputs
All charts are saved automatically in:
- `images/`

## How to Run
```bash
cd sales-analysis/src
python sales_analysis.py
